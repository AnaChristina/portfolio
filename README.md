### 📱 Aplicação de Atendimento ao Cliente (IACIT)
Aplicação mobile responsiva de atendimento ao cliente, com registro de ocorrências e comunicação em tempo real via chat entre usuário e administrador.

Atuei no desenvolvimento de interfaces para uma aplicação mobile responsiva de atendimento ao cliente, permitindo o registro e acompanhamento de ocorrências (ROs), além de comunicação via chat entre usuário e administrador.
Participei prototipação e desenvolvimento das telas utilizando Figma, React Native e TypeScript, desenvolvendo funcionalidades como listagem, filtragem e detalhamento de registros, além da interface de chat.
Contribuí na integração entre front-end e back-end na funcionalidade de registro de ocorrências, participando da estruturação da comunicação com a API.
Participei do desenvolvimento em equipe utilizando metodologia ágil, colaborando com organização de tarefas e acompanhamento do backlog.
A aplicação também considerou requisitos de privacidade de dados, incluindo funcionalidades como consentimento de uso (opt-in/opt-out) e exclusão de dados do usuário.

🔧 Destaques:
- Registro e acompanhamento de chamados
- Sistema de filtragem por status
- Chat em tempo real
- Funcionalidades de privacidade de dados (LGPD)

💻 Tecnologias:
React Native, TypeScript, Node.js, MongoDB
